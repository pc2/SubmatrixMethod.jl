Benchmarking

The benchmarking facilities are turned off entirely by default to avoid any performance overhead.

julia> using SubmatrixMethod
julia> SubmatrixMethod.enable_benchmarks()[ Info: Benchmarks enabled!
julia> A = SubmatrixMethod.generate_input_matrix(1000, 0.01);
julia> submatrix_apply(inv, A);
julia> submatrix_apply(inv, A);
julia> SubmatrixMethod.print_benchmarks() ──────────────────────────────────────────────────────────────────────────────

Benchmarks can be reset by calling SubmatrixMethod.reset_benchmarks!() or turned off again via SubmatrixMethod.disable_benchmarks().

Note: Only the serial variant, i.e. with multithreading=false (default), can be benchmarked for now.
