SubmatrixMethod.jl

Approximately compute matrix functions of large sparse matrices by using the submatrix method.

Technically, we support arbitrary positive definite input matrices but the approximation / performance is best for (very) sparse matrices (A::SparseMatrixCSC).

Installation

You can simply add SubmatrixMethod.jl to your Julia environment with the command

] add https://git.uni-paderborn.de/pc2/julia/submatrixmethod.jl
Note

The minimal required Julia version is 1.6 but we recommend using Julia ≥ 1.7.

TLDR

using SubmatrixMethod

# sparse, symmetric, random, positive definite matrix of
# size 1000 x 1000 with density around 0.05.
A = SubmatrixMethod.generate_input_matrix(1000, 0.05)

X = inv(Matrix(A))
Y = submatrix_apply(inv, A)

maximum(abs.(X .- Y) ≤ 1e-7 # true
